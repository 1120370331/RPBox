# SRV-0005: 数据库自动备份

| 属性 | 值 |
|------|-----|
| ID | SRV-0005 |
| 类型 | OPS |
| 优先级 | P1 |
| 状态 | DONE |
| 模块 | server/internal/backup |
| 依赖 | - |

---

## 1. 目标

- 每小时自动备份 PostgreSQL 数据库
- 备份文件上传到 OSS
- 开发/生产环境隔离
- 备份保留 30 天

---

## 2. 验收标准

- [x] 定时生成 pg_dump 备份
- [x] OSS 上传成功，按环境前缀隔离
- [x] 备份保留 30 天
