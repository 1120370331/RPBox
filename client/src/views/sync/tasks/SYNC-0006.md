## 1. 目标

管理本地同步状态，检测变更。

**版本标识参考**: PRD2 - 5.7 版本标识机制

---

## 2. 接口设计

```rust
pub struct SyncMetadata {
    pub profile_id: String,
    pub last_synced_at: DateTime<Utc>,
    pub last_synced_checksum: String,
    pub cloud_version: i32,
}

pub fn get_sync_status(profile_id: &str) -> SyncStatus;
pub fn update_sync_metadata(metadata: SyncMetadata);

pub enum SyncStatus {
    Synced,      // 已同步
    LocalChanged,  // 本地有变更
    CloudChanged,  // 云端有变更
    Conflict,    // 冲突
}
```
