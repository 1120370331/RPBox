# 项目介绍
这是我用于学习+开源的项目
旨在服务于魔兽世界RP活动玩家
魔兽世界RP玩法依赖TotalRP3与TotalRP3 Extended插件
https://github.com/Total-RP/Total-RP-3.git
https://github.com/Total-RP/Total-RP-3-Extended.git
解决以下几个痛点。
1. 人物卡跨设备备份加载难
现有的魔兽插件软件的，大多聚焦插件目录的备份
但TRP3的插件数据存在./WTF目录下
这导致了大多数插件软件无法备份并上云，且无法在云端查看与编辑管理。

2. RP剧情记录难
魔兽的一场RP将包含大量文字对话与场景描述内容，这些都是玩家创作的心血，而魔兽世界与历史记录插件都没有做到长期归档并保存这些记录。
3. 社区分享交流难
现在的“活动战报/剧情总结”通常是在百度贴吧金色平原吧或者NGA论坛进行展示，没有按公会/人物/剧情线归档，没有便利的检索机制，导致大家难以相互借鉴，好的剧情难以被大家看到。
4. TRP3道具获取难
TRP3提供了道具制作功能，现在唯一的分享途径是weakauras的官网，作为一个子功能，但是，这个网国内需要翻墙，且他们主要还是做weakauras脚本的，并没有针对TRP3进行调优检索和社区推荐，很难找到适合的道具。

综上所述，本软件主张以轻量化的方案解决上述问题，并内置各类小工具、AI功能来辅助魔兽世界RP剧情进行。他可以是插件+软件一体化的平台。
