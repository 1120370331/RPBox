# 贴吧文风指南

> 参考来源：金色平原吧《角色扮演指南》

## 文风特点

### 1. 标题格式
- 使用【】标注帖子类型：`【角色扮演指南】扮演资料导航及新人科普站`
- 简洁明了，说明帖子用途

### 2. 正文结构
- **前言**：先说背景和目的，拉近距离
- **目录**：列出大纲，方便阅读
- **分隔线**：用 `······························` 或 `——————————★` 分隔章节
- **层级清晰**：一、二、三 + （一）（二）（三）

### 3. 语气特点
- 正式但不失亲切
- 像老玩家带新人的口吻
- 用"你"直接称呼读者
- 举具体例子说明抽象概念

### 4. 内容特点
- 详实，不怕长
- 大量引用链接（贴吧帖子、NGA帖子）
- 图文结合
- 有"补楼"机制（在回复中补充内容）

### 5. 互动元素
- 欢迎讨论
- 引导回复
- 资源汇总便于收藏

## 活动/通知体文风（参考：比武大会帖）

> 适用于：功能发布、版本更新、活动通知等"半通知"类型

### 核心公式

```
随意开场 + 核心信息 + 有梗收尾
```

### 1. 开头风格

**不要**：冷冰冰的公告
```
RPBox v1.0 发布，新增以下功能...
```

**不要**：太煽情
```
感谢各位依然在此，在这艾泽拉斯继续书写我们的故事...
```

**要**：随意 + 简短RP腔调
```
各位艾泽拉斯的记录者们，鸽了这么久终于更新了。
```
```
诸位冒险者，这次带来的是xxx功能。
```
```
致各位故事的书写者：最近肝了个新功能，来看看。
```

**技巧**：
- 开头可以带简短的RP称呼（记录者们、冒险者、书写者）
- 正文随意，像在群里发消息
- 可以自嘲（鸽了、摸鱼、终于肝完了）
- 不要长篇抒情，点到为止

### 2. 信息结构（清晰但不死板）

```
【核心信息区】
功能/活动：xxx
时间：xxx
地点/链接：xxx

【亮点/奖励区】
· 亮点1：xxx
· 亮点2：xxx
· 亮点3：xxx

【使用方式/参与方式】
1. 第一步
2. 第二步
3. 第三步
```

### 3. 幽默元素（灵魂所在）

**自嘲式幽默**：
> "我不明白，为什么还有人再提不会参加比赛...如果再没学会，就去闪金镇蹲下和熊猫人一起跪着"

**梗式收尾**：
> "总之比武大会，优势在窝。开赛！"

**吐槽式说明**：
> "要是你还不会用，那我也没办法了，建议去奥格瑞玛门口罚站"

### 4. 背景故事（加分项）

- 附带历史/背景介绍，增加厚重感
- 体现"用心"和"专业"
- 让读者觉得"这人是真懂"

### 5. 分楼发布

- 1楼：核心信息 + 感情开场
- 2楼：详细说明/教程
- 3楼：背景故事（可选）
- 4楼：常见问题/补充

## 完整示例

### 示例1：项目介绍帖（首发帖）

```
【工具分享】RPBox - 给RP玩家做的工具箱，解决TRP3几个老大难问题

各位艾泽拉斯的记录者们，我是一个玩了很多年RP的老玩家。

最近自己搞了个工具，想分享给大家。先说说为什么要做这个东西。

······························

一、痛点

玩RP这么久，有几个问题一直困扰我：

（一）人物卡不能云同步
换电脑、重装系统，精心写的人物卡就没了。
每次都要手动备份SavedVariables，麻烦得要死。
有一次硬盘坏了，三年的人物卡全没了，心态直接崩了。

（二）剧情记录找不到
和朋友们跑的剧情，过几天就找不到了。
聊天记录一刷新就没，只能靠截图。
截图多了又乱，想找某段剧情得翻半天。

（三）好东西没地方分享
做了个自己觉得还不错的TRP3道具，想分享给别人。
发贴吧？格式乱七八糟。发群里？过几天就被刷没了。
没有一个专门的地方能好好展示和分享。

······························

二、RPBox 是什么

简单说就是一个桌面软件 + 配套插件，专门解决上面这些问题。

目前做了这些功能：
· 人物卡云备份同步 - 再也不怕丢档了
· 剧情自动归档 - 插件自动记录，客户端整理回放
· 社区分享 - 以后会做，可以分享人物卡和道具

开源免费，代码在GitHub上，不收钱不打广告。

······························

三、怎么用

1. 下载RPBox客户端（链接在最后）
2. 把配套插件丢到AddOns文件夹
3. 打开客户端，登录，完事

具体教程我后面会单独发一个帖子。

······························

【下载地址】
GitHub: https://github.com/xxx/RPBox
蓝奏云: https://xxx.lanzou.com/xxx （密码：rpbox）

有问题可以楼下问，我看到会回。
也欢迎提建议，想要什么功能可以说说。

对了，项目刚起步，bug肯定有，别骂太狠就行。
```

## 对比：小红书 vs 贴吧

| 维度 | 小红书 | 贴吧 |
|------|--------|------|
| 长度 | 短，300字内 | 长，可以几千字 |
| 语气 | 活泼，多emoji | 正式但亲切，有梗 |
| 结构 | 碎片化 | 层级分明 |
| 图片 | 封面很重要 | 内容配图 |
| 互动 | 评论区简短 | 楼中楼讨论 |
| 幽默 | 网感、流行语 | 圈内梗、自嘲 |
