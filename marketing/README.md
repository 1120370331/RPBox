# RPBox 宣传资料库

## 目录结构

```
marketing/
├── devlog/          # 开发日记
│   ├── 001-xxx.md   # 按序号命名
│   └── ...
├── assets/          # 图片、视频等素材
│   ├── screenshots/ # 截图
│   └── covers/      # 封面图
└── README.md        # 本文件
```

## 发布平台

- 小红书
- 百度贴吧 (魔兽世界吧、RP相关吧)
- NGA 玩家社区

## 开发日记命名规范

`[序号]-[主题].md`

示例：
- `001-项目启动.md`
- `002-人物卡同步功能.md`
- `003-剧情归档设计.md`

## 内容模板

每篇开发日记建议包含：
1. 标题（吸引眼球）
2. 本期亮点（1-3个）
3. 开发进展（配图）
4. 下期预告
5. 互动话题
