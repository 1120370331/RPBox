## 正文

```
【工具分享】RPBox - 给RP玩家做的工具箱，诚邀各位来测试

各位艾泽拉斯的记录者们，我是一个玩了好几年RP的老玩家。

最近自己搞了个工具叫RPBox，目前做了两个功能：人物卡云备份和剧情自动记录。
东西基本能用了，想找人帮忙测测，顺便听听大家的意见。

······························

一、为什么做这个

玩RP这么久，有几个问题一直困扰我：

（一）人物卡不能云同步

换电脑、重装系统，精心写的人物卡就没了。
每次都要手动备份SavedVariables，麻烦得要死。
之前硬盘坏过一次，好几年的人物卡全没了，当时心态直接崩了。

（二）剧情记录找不到

和朋友们跑的剧情，过几天就找不到了。
聊天记录一刷新就没，只能靠截图。
截图多了又乱，想找某段对话得翻半天。

所以我就想，能不能自己做个工具解决这些问题。

······························

二、RPBox 目前能干啥

现在做好的功能有两个：

【人物卡云备份】
· 自动读取你的TRP3人物卡数据
· 上传到云端，换电脑也能同步回来
· 支持多角色管理

【剧情自动记录】
· 配套插件自动记录RP对话（说、喊、表情、小队等）
· 客户端整理成时间线，可以回看
· 显示参与者的TRP3角色名
· 支持搜索和导出

[图片位置：剧情归档界面截图]

······························

三、怎么用

1. 下载RPBox客户端
2. 把配套插件（RPBox_Addon）丢到Interface/AddOns文件夹
3. 打开客户端，注册登录
4. 进游戏，插件会自动工作

【下载地址】
GitHub: [链接]
蓝奏云: [链接]

······························

四、找人测试

现在功能基本能用，但肯定还有bug和不完善的地方。

希望有兴趣的朋友帮忙测测，反馈一下：
· 有没有报错、闪退
· 哪里用着不顺手
· 想要什么功能

项目开源免费，不收钱不打广告，纯粹为爱发电。
代码都在GitHub上，感兴趣的可以看看。

······························

有问题可以楼下问，我看到会回。
骂轻点，毕竟是个人项目，精力有限。

后续打算做：道具分享、社区功能
```

### 2楼：详细安装教程

```
【详细安装教程】

一、下载

客户端：
· GitHub: [链接]
· 蓝奏云: [链接]

插件：
· 在客户端的"设置-插件管理"里可以直接下载
· 或者手动下载：[链接]

二、安装插件

1. 找到你的魔兽世界安装目录
2. 进入 _retail_/Interface/AddOns（正式服）
   或 _classic_/Interface/AddOns（怀旧服）
3. 把 RPBox_Addon 文件夹丢进去
4. 重启游戏，在插件列表里勾选启用

三、使用客户端

1. 打开RPBox客户端
2. 注册账号（邮箱验证）
3. 登录后，设置你的WoW安装路径
4. 完事，功能都在左边菜单里

有问题这楼问。
```

### 3楼：常见问题

```
【常见问题】

Q：支持什么服务器？
A：目前测试的是国服正式服，怀旧服理论上也行但没测过。

Q：会不会被封号？
A：不会。这个工具只是读取本地的SavedVariables文件，不修改游戏内存，不自动操作，跟你手动备份没区别。

Q：数据安全吗？
A：人物卡数据上传到我自己的服务器，不会分享给第三方。代码开源，不放心可以自己看。

Q：能不能离线用？
A：剧情记录功能需要插件在线记录，但回看可以离线。人物卡备份需要联网同步。

Q：以后会收费吗？
A：不会。这是个人项目，为爱发电。

其他问题楼下问。
```
