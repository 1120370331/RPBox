### 1.3 安装教程

**步骤说明**
1. 下载安装包（RPBox_Setup.exe）
2. 双击运行安装程序
3. 按照向导完成安装
4. 启动RPBox并登录账号
5. 开始使用完整功能

**注意事项**
- Windows Defender可能提示安全警告，选择"仍要运行"
- 首次启动需要联网激活
- 建议关闭魔兽世界后再安装

## 三、数据接口需求

### 3.1 获取最新版本信息
```
GET /api/v1/updater/latest
返回：
{
  version: string,
  release_date: string,
  file_size: number,
  download_url: string,
  release_notes: string
}
```

### 3.2 获取版本历史
```
GET /api/v1/updater/versions
返回：所有历史版本列表
```

### 3.3 下载统计
```
POST /api/v1/updater/download-count
记录下载次数（用于统计）
```

### 4.1 下载失败
- 提供备用下载链接（GitHub Releases）
- 显示错误提示

- 显示"您已安装 v1.1.0，最新版本 v1.2.0"
- 提供更新按钮

### 4.3 不支持的系统
- 检测用户操作系统
- 如果是Mac/Linux，显示"暂不支持该系统"
- 提供邮件订阅，新系统支持时通知
