## 二、系统配色方案

### 2.1 主色调
```css
/* 主色系 */
--color-primary: #4B3621      /* 深棕色 - 主要文字 */
--color-accent: #B87333       /* 铜色 - 按钮、强调 */
--color-secondary: #856a52    /* 中棕色 - 次要文字 */

/* 背景色 */
--color-bg-page: #EED9C4      /* 羊皮纸色 - 页面背景 */
--color-bg-hero: linear-gradient(135deg, #4B3621, #856a52) /* Hero区域渐变 */
--color-bg-card: #fff         /* 白色 - 功能卡片背景 */
--color-bg-feature: #f5f0eb   /* 浅米色 - 特色卡片背景 */

/* 边框色 */
--color-border: #e5d4c1       /* 浅边框 */
--color-border-dark: #d4a373  /* 深边框 */
```

### 2.2 功能色
```css
/* 按钮 */
--color-btn-primary: #B87333     /* 主按钮 */
--color-btn-primary-hover: #8B5A2B /* 主按钮悬停 */
--color-btn-secondary: #856a52   /* 次按钮 */

/* 状态色 */
--color-success: #28a745      /* 成功 */
--color-info: #3498db         /* 信息 */
--color-warning: #f57c00      /* 警告 */
```

### 2.3 文字色
```css
--text-primary: #4B3621       /* 主要文字 */
--text-secondary: #856a52     /* 次要文字 */
--text-tertiary: #a89984      /* 辅助文字 */
--text-hero: #fff             /* Hero区域文字（白色） */
--text-link: #B87333          /* 链接文字 */
--text-link-hover: #8B5A2B    /* 链接悬停 */
```

## 三、数据接口需求

### 3.1 获取平台统计数据
```
GET /api/v1/stats/platform
返回：
{
  user_count: number,
  character_count: number,
  story_count: number,
  item_download_count: number
}
```

### 3.2 获取最新版本信息
```
GET /api/v1/updater/latest
返回：版本号、下载链接、文件大小
```

## 四、性能要求

- 首屏加载时间 < 1.5秒
- 图片使用WebP格式，懒加载
- Hero区域背景图优化
- 功能卡片动画流畅（60fps）
- 移动端优先加载关键内容

## 五、其他说明

**移动端适配**
- Hero区域高度调整
- 功能卡片改为2列或1列
- 按钮尺寸增大，便于点击
- 导航改为汉堡菜单

**SEO优化**
- 页面标题：RPBox - 魔兽世界RP玩家的云端工具箱
- Meta描述：人物卡同步、剧情归档、社区分享、道具市场，为魔兽世界RP玩家打造的完整解决方案
- 结构化数据标记（WebSite, Organization）
- Open Graph标签（社交分享）

**动画效果**
- Hero区域淡入动画
- 功能卡片依次出现
- 数字滚动动画
- 悬停效果流畅过渡

**无障碍**
- 所有图片添加alt属性
- 按钮支持键盘操作
- 合理的标题层级（H1-H6）
- 高对比度文字
