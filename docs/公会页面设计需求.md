# 公会页面设计需求文档

## 📋 页面概述

创建两个独立的公会专属页面：
1. **公会帖子页面**（GuildPosts.vue）- 展示公会相关的所有帖子
2. **公会剧情页面**（GuildStories.vue）- 展示公会归档的剧情记录

**设计风格：** 中世纪奇幻风格，温暖的棕色系，古朴典雅

---

## 1️⃣ 公会帖子页面（GuildPosts）

### 功能需求

#### 页面头部
- 返回按钮（返回公会详情页）
- 页面标题：`{公会名称} - 帖子`
- 描述文案：`查看公会成员发布的帖子`

#### 筛选和搜索区域
- **分类筛选按钮组**：全部、人物卡、公会、战报、小说、道具、活动、其他
- **排序下拉框**：最新发布、热门讨论、最多浏览
- **搜索输入框**：搜索帖子标题或内容
- **帖子计数**：显示 `共 X 个帖子`

#### 帖子列表
- **卡片式布局**，每个帖子卡片包含：
  - 分类标签（左上角）
  - 置顶/精华标记（如果有）
  - 帖子标题（18px，加粗）
  - 帖子摘要（2行截断）
  - 作者信息：头像 + 用户名 + 发布时间
  - 统计数据：浏览数、评论数

#### 分页
- 上一页/下一页按钮
- 页码显示：`第 X / Y 页`

#### 状态处理
- 加载状态：旋转图标 + "加载中..."
- 空状态：图标 + "暂无帖子" + 说明文字
- 搜索无结果：图标 + "未找到匹配的帖子"

---

## 2️⃣ 公会剧情页面（GuildStories）

### 功能需求

#### 页面头部
- 返回按钮（返回公会详情页）
- 页面标题：`{公会名称} - 剧情归档`
- 描述文案：`查看公会成员归档的剧情记录`

#### 搜索区域
- **搜索输入框**：搜索剧情标题或描述
- **剧情计数**：显示 `共 X 个剧情`

#### 剧情列表
- **列表式布局**，每个剧情卡片包含：
  - 剧情标题（18px，加粗）
  - 剧情描述（2行截断）
  - 发布日期
  - 统计数据：记录条数、持续时间

#### 状态处理
- 加载状态：旋转图标 + "加载中..."
- 空状态：图标 + "暂无剧情归档"
- 搜索无结果：图标 + "未找到匹配的剧情"

## 🎨 配色方案

### 主色系
- **深棕色** `#2C1810` - 主标题、重要文字
- **棕红色** `#804030` - 主要按钮、强调元素
- **金棕色** `#B87333` - 悬停状态、边框高亮
- **浅金棕** `#D4A373` - 徽章、标签

### 辅助色系
- **中灰棕** `#8D7B68` - 次要文字、说明文字
- **深灰棕** `#4B3621` - 正文文字
- **浅灰棕** `#8C7B70` - 禁用状态、占位符

### 背景色系
- **页面背景** `#EED9C4` - 页面主背景
- **卡片背景** `#FFFFFF` - 卡片、弹窗
- **浅色背景** `#F5EFE7` - 搜索框、标签
- **悬停背景** `#FFF5E6` - 悬停效果

### 边框色系
- **主边框** `#E5D4C1` - 卡片边框、输入框边框
- **分割线** `#F5EFE7` - 内容分割线
- **高亮边框** `#B87333` - 聚焦状态、选中状态

### 功能色系
- **成功绿** `#16A34A` - 成功提示
- **警告橙** `#D97706` - 警告提示、活动标记
- **错误红** `#DC2626` - 错误提示、删除操作
- **信息蓝** `#3B82F6` - 信息提示

### 特殊标记
- **置顶标签** 背景 `#804030` / 文字 `#FFFFFF`
- **精华标签** 渐变 `linear-gradient(135deg, #E6A23C, #D97706)` / 文字 `#FFFFFF`

### 阵营色系
- **联盟** 背景 `#EBF5FF` / 文字 `#1D4ED8` / 边框 `#BFDBFE`
- **部落** 背景 `#FEE2E2` / 文字 `#DC2626` / 边框 `#FECACA`
- **中立** 背景 `#F5EFE7` / 文字 `#B87333` / 边框 `#E5D4C1`

## 📝 补充说明

### 交互效果
- 卡片悬停：边框变为高亮边框色，轻微上移（2px），增加阴影
- 按钮悬停：背景色加深，光标变为手型
- 输入框聚焦：边框变为高亮边框色，添加外发光效果

### 响应式要求
- 页面最大宽度：1200px
- 卡片间距：16px
- 内边距：20-24px

### 字体规范
- 标题字体：18px，加粗（font-weight: 600）
- 正文字体：14px，常规（font-weight: 400）
- 说明文字：12-13px，常规
