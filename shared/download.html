简单的开始方式

1

下载安装包

点击上方的下载按钮获取 .exe 文件。文件大小约 145MB。

2

运行与安装

双击运行。如遇 Windows 安全提示，请点击"更多信息"后选择"仍要运行"。

3

登录并同步

使用 Web 版账号登录，您的人物卡和收藏将自动同步至本地。

常见问题

安装失败或报错怎么办？
请确保您已关闭杀毒软件或将安装包添加至白名单。如果是网络问题，请尝试备用下载链接。
支持 Mac 或 Linux 吗？
目前仅支持 Windows 10/11。Mac 版本正在开发中，预计将于 Q4 发布测试版，敬请期待。
客户端占用多少空间？
客户端体积很小（Tauri 架构），通常 20MB 左右即可运行。
